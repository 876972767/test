4、简答题
286)、Android 开发环境（Android Studio）需安装什么？
正确答案：Java JDK：Java 开发环境，需配置环境变量； Android Studio：
Android Studio 是开发 Android 应用程序的官方 IDE，基于 Intellij IDEA；
Android SDK：对应版本的 Android 软件开发包;
287)、res 文件夹的作用是什么？其有什么特殊性？
正确答案：res 文件夹可存放项目的资源文件夹。如图片、文本等。 res 文件夹下
的所有资源，都会在 R.java 文件下生成对应的资源 ID（32 位二进制）。
288)、在布局文件中表示资源 ID 的方式是？
正确答案：基本格式为：属性=“@****/*****” 如
android:drawableTop="@drawable/tab_menu_channel" 如
android:textColor="@color/tab_menu_text" 
289)、在 java 文件中表示资源 ID 的方式是？
正确答案：基本格式：R.*****.******(R.资源类型.资源名) 如
setContentView(R.layout.activity_main); 
290)、xml 文件和 java 文件是怎么关联的？
正确答案：setContentView(R.layout.activity_main); 
291)、如何给控件添加 ID 的别名？
正确答案：android:id="@+id/submit"
292)、如何在 java 文件中初始化控件？
正确答案：先声明对象的类型和名字； 用 findViewByID 方法初始化对象；参数：
R.id.布局文件中标记的别名；
293)、请列举一些 TextView 控件的常用属性。
正确答案：android:text 设置显示的文本内容 android:textSize 设置字体大小
android:textColor 设置字体颜色 android:background 设置背景颜色
294)、如何在 java 文件中修改 TextView 的显示文本？
正确答案：根据布局文件中所需的属性，寻找对应的 java 方法。 修改显示的文本
对应属性:android:text，其对应方法为 setText();
295)、如何在 java 文件中让 RadioGroup 在界面中不可见？
正确答案：radioGroup.setVisibility(View.INVISIBLE); 
296)、列举几种常见的消息输出方式。
正确答案：Toast 类：以短小信息的形式通知给用户，在一段时间之后会自动消
失。Log 类：Android Studio 中的日志工具类。Log 可以通过过滤器来进行筛选，
方便用户查看。
297)、新建 Activity 时，默认会新建哪 2 个文件？作用是什么？
正确答案：布局文件：activity_main.xml，决定界面显示效果 Java 文件：
MainActivity.java, 实现逻辑功能
298)、描述使用匿名内部类的方式实现按钮单击事件的过程。
正确答案：初始化按钮控件；按钮控件调用 setOn*****Listener()方法实现单击事
件；
299)、什么是 Activity？
正确答案：Activity 是 Android 的四大组件之一，它是一种可以包含用户界面的组
件，主要用于和用户进行交互。 Activity 用于显示用户界面，用户通过 Activity 交
互完成相关操作 ， 一个 App 允许有多个 Activity。
300)、什么是 Activity 的生命周期？Activity 的生命周期共有几种状态？
正确答案：表示一个 Activity 从开始运行到最后销毁的过程。 Resumed（运行状
态） Paused（暂停状态） Stopped（停止状态） Destroy（销毁状态）
301)、Activity 有几种回调方法，各是什么？
正确答案：共有 7 种回调方法。 onCreate() onStart() onResume() onPause() 
onStop() onDestory() onRestart()
302)、如何启动新的 Activity？
正确答案：在 Activity 与 Activity 中相互通信，即为 Android 的组件间的相互通
信，必须通过 Intent 对象完成。 初始化 Intent 对象，设置当前界面和需启动的界
面； 通过 StartActivity 方法启动界面；
303)、SharedPreferences 的数据是什么形式，存在哪儿？
正确答案：Xml 文件的形式，存放在 data/data/包名/shared_prefs
304)、SQLite 数据库存放路径是？
正确答案：data/data/包名/database
305)、调试 SQLite 数据库可用什么工具？
正确答案：可用 SDK 目录下 platform-tools 中的 sqlite3.exe
