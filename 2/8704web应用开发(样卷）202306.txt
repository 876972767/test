综合题(共 1 题,共 10 分)
46.(10 分)编写一个学校信息的 JavaBean，取名 SchoolBean。要求如下：
（1）定义一个包，将该 bean 编译后生成的类存入 com.bean 包中；
（2）设计三个属性学校名称（schoolName，字符串型）、地址（address，字符串型）和是
否示范学校（demonstrate，布尔型）；
（3）设计访问属性的相应方法。
请按照 JavaBean 的规范编写程序。
参考答案：
package com.bean;
public class SchoolBean { //给出包及属性定义
 private String schoolName; //学校名称
 private String address; //学校地址
 private boolean demonstrate; //是否示范性学校 
 public String getSchoolName() { //给出属性的 set、get 方法
 return schoolName;
 }
 public void setSchoolName(String schoolName) {
 this. schoolName = schoolName;
 }
 public String getAddress() { //给出属性的 set、get 方法
 return address;
 }
 public void setAddress (String address) {
 this. address = address;
 }
 public boolean isDemonstrate () { //给出属性的 set、get 方法
 return demonstrate;
 }
 public void setDemonstrate (boolean demonstrate) {
 this. demonstrate = demonstrate;
 }
}
